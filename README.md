## 田径场训练
在深圳这个城市从来不缺田径场地，最常去的就是福田和宝安的体育馆，大部分学校也都可以预约进入～

1、有氧训练
过多的有氧训练会限制无氧以及爆发力的发展，但是在前期是可以进行一些有氧训练的，不仅可以锻炼韧带的强度，也可以控制一些脂肪，时间控制在25分钟上下，速度可以稍微放慢一些，每周我会至少进行1次或者2次的慢跑，也是为了以后的冲刺跑作准备～
